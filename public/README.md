# Public Assets - SEO Files

This folder contains important SEO and PWA files.

## Files Included:
- ✅ `robots.txt` - Search engine crawler instructions
- ✅ `sitemap.xml` - Website structure for search engines
- ✅ `site.webmanifest` - PWA manifest for mobile devices

## Missing Favicon Files (Perlu ditambahkan):

Anda perlu menambahkan file-file favicon berikut ke folder `public/` agar website terlihat lebih profesional:

- `favicon.ico` (16x16, 32x32, 48x48)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180x180)
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`

### Cara Membuat Favicon:

1. **Option 1: Generator Online**
   - Kunjungi: https://realfavicongenerator.net/
   - Upload logo/gambar Anda
   - Download semua file yang dihasilkan
   - Paste ke folder `public/`

2. **Option 2: Manual dengan Canva**
   - Buat desain logo 512x512px di Canva
   - Export sebagai PNG
   - Gunakan tool seperti favicon.io untuk generate semua ukuran

3. **Option 3: Simple Placeholder**
   - Buat file PNG sederhana dengan initial "JBW"
   - Resize ke berbagai ukuran yang diperlukan

## Update Sitemap

Jangan lupa untuk update `lastmod` di sitemap.xml setiap kali ada perubahan besar di website.
